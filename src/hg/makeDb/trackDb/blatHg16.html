Description

This track displays alignments of the $o_date $o_organism genome vs. the $organism genome.

Methods

The alignments were made with blat in translated protein mode using the parameters -q=dnax -t=dnax. The default settings were used otherwise. Both genomes were masked with RepeatMasker and Tandem Repeat Finder before running blat.

Credits

This track was generated at UCSC.

References

Kent, W.J. BLAT - the BLAST-like alignment tool. Genome Res. 12(4), 656-664 (2002).
