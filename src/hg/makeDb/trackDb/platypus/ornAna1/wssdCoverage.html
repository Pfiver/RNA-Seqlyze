Description

This track represents coverage of clones that were assayed for segmental duplications using high-depth sequence reads from the Whole Genome Shotgun assembly of $Organism. Regions absent from this track were not assessed by this version of the Segmental Duplication Database (SDD). For a description of the whole-genome shotgun sequence detection (WSSD) method, see Bailey JA et al (2002) in the References section below.
